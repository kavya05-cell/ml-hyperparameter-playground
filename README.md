# ML Hyperparameter Playground

An interactive platform to experiment with machine learning algorithms and understand how hyperparameters affect model performance.

## Features

- Train ML models on custom datasets
- Adjust hyperparameters interactively
- Compare model performance
- Visualize metrics and decision boundaries

## Algorithms

- Logistic Regression
- K-Nearest Neighbors
- Random Forest

## Tech Stack

Frontend: React / Streamlit  
Backend: FastAPI  
ML: Scikit-learn  
Visualization: Plotly

## Project Goals

This project aims to build an experimentation platform where users can understand how machine learning models behave under different hyperparameter configurations.
